In order to create/update tables, there are a few commands that should be executed in order:

[x] delete Migrations folder in JobSearch.Data project
[x] dotnet ef database drop --force [in DataScraper project]
[x] dotnet ef migrations add InitialCreate -p JobSearch.Data -s JobSearch.DataScraper
[x] dotnet ef database update -p JobSearch.Data -s JobSearch.DataScraper

To run scrapers locally:

[x] dotnet user-secrets set --project JobSearch.DataScraper "USAJobs:ApiKey" "your-actual-api-key"
[x] dotnet user-secrets set --project JobSearch.DataScraper "USAJobs:Email" "your-email@example.com"
[x] dotnet user-secrets set --project JobSearch.DataScraper "CareersInPoland:Email" "your-email@example.com"

To set a scraper to run locally:

[x] Go to JobSearch.DataScraper/Properties/launchSettings.json
[x] Set "commandLineArgs": "--scraper <current-scraper>",
